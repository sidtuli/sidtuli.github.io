About

Hello there! My name is Sidhanth Tuli, but you can call me Siddy. I'm a junior computer science student at the University of Maryland, College Park. I've dabbled in a wide range of areas, but I would say I've mainly concentrated on frontend web development. I'm currently a web developer at the Diamondback Lab on my campus. I'm a part of Terrapin Hackers, the UMD Math Club, and WMUC.

Projects

Barrier

Created an interactive film project about censorship and internet access over the course of a semester as a part of Honors College program.Used popcorn.js for video streaming and jQuery to work with navigation and map functionality.

InformedApp

Interned at a government contractor to develop a hybrid and mobile app using PhoneGap and jQuery. The app searches the in-beta OpenFDA API to deliver information on medicine for consumers.

Map Games

My own personal side project of several map quizzes to allow people to learn geographical data. Made using the jQuery Vector Map plugin for map interactivity, Geonames API for capital information and flags, and wikipedia parsing to get state capitals.

You can check out my Github page to see more of my work as well as see miscellaneous pieces of work I've done on my "random page."
